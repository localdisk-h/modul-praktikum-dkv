PAKET MODUL PRAKTIKUM DKV BERBASIS HTML

Cara menggunakan:
1. Ekstrak file ZIP.
2. Buka index.html melalui Google Chrome, Microsoft Edge, atau Firefox.
3. Semua modul berjalan tanpa internet.
4. Progres tersimpan pada browser menggunakan localStorage.
5. Untuk membagikan ke komputer lain, salin seluruh folder.

Isi:
- index.html
- panduan-guru.html
- modul-1-warna.html
- modul-2-tipografi.html
- modul-3-komposisi.html
- modul-4-fotografi.html
- modul-5-design-brief.html
- styles.css
- common.js

Paket ini bebas dimodifikasi untuk kebutuhan pembelajaran internal sekolah.
