Cara menggunakan modul di kelas

Modul dirancang untuk pembelajaran berbasis proyek. Guru dapat menampilkan modul melalui proyektor, membagikan folder ke komputer siswa, atau menaruhnya di server lokal sekolah.

Alur satu pertemuan

TahapDurasiAktivitas
Pembukaan10 menitGuru menjelaskan tujuan, konteks proyek, dan indikator keberhasilan.
Eksplorasi20 menitSiswa mencoba simulasi dan mengamati perubahan visual.
Praktik60 sampai 120 menitSiswa menyelesaikan tugas proyek sesuai brief.
Presentasi20 menitSiswa menunjukkan hasil dan menjelaskan keputusan desain.
Refleksi10 menitSiswa mengerjakan kuis atau menulis refleksi.

Rubrik umum

AspekBobotIndikator
Pemahaman konsep20%Keputusan visual sesuai prinsip yang dipelajari.
Proses kerja20%Langkah kerja runtut, file tertata, dan waktu terkendali.
Keterampilan teknis25%Penggunaan alat atau aplikasi tepat.
Kualitas hasil25%Komunikatif, rapi, konsisten, dan sesuai brief.
Presentasi dan refleksi10%Mampu menjelaskan keputusan dan menerima umpan balik.

Teknis penggunaan

  1. Ekstrak folder ZIP.
  2. Buka file index.html menggunakan Chrome, Edge, atau Firefox.
  3. Salin seluruh folder ke setiap komputer siswa.
  4. Progres tersimpan di browser masing-masing melalui localStorage.
  5. Gunakan tombol cetak pada modul Design Brief untuk menghasilkan dokumen PDF.
